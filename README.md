# Melvin-s-Wheels-and-Deals
Overview
Melvin's Wheels and Deals is a comprehensive web application designed for car sales, offering users an intuitive platform to browse, buy, and sell vehicles. Our website features a diverse inventory of new and used cars, making it easy for potential buyers to find their ideal vehicle. Sellers can list their cars conveniently, ensuring a seamless transaction process.

Features
Car Listings:

Extensive listings of new and used cars with detailed specifications, images, and pricing information.
Advanced search and filtering options to help users find cars based on make, model, price, mileage, and other criteria.
Vehicle Details:

In-depth details for each listing, including engine specifications, mileage, condition, and seller contact information.
User Accounts:

Users can create accounts to save favorite listings, manage their listings, and track their inquiries.
Seller Dashboard:

Sellers can easily list their vehicles, manage their listings, and communicate with potential buyers through the platform.
Responsive Design:

Fully responsive website, ensuring an optimal viewing experience across desktops, tablets, and mobile devices.

Installation
Clone the repository:

bash
git clone 

Navigate to the project directory:

bash
cd melvins-wheels-and-deals

Ensure you have XAMPP installed. Start the Apache and MySQL modules.

Import the database:

Open phpMyAdmin via XAMPP.
Create a new database named melvins_wheels.
Import the provided SQL file (if available) to set up the necessary tables.
Run the application:

Launch your Python environment and run the Flask application.
Open index.html in your web browser to start using the website.
